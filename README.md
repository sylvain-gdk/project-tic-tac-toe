# Tic-Tac-Toe game

## About the project

This is just a simple and fun version of the Tic-Tac-Toe game that allows one player to play against the computer. It also keeps stats.

## Dependencies

- JavaSDK 1.8
- JavaSE Runtime 1.8 (to execute the compiled project)
